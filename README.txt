Задача:

Сделать   тестовое  приложение, которое будет отображать пользователю погоду с возможностью вносить и изменять информацию.

Функционал:

- Клиентская часть реализована на WPF

- Возможность корректировать погоду;

- Отображение погоды для пользователя на неделю;

Используемые технологии:

С# / WPF /
